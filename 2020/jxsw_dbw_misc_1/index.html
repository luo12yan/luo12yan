葫芦娃

最先看到的是一堆文件和一个MP3

所谓葫芦七兄弟嘛，把那七个文件合并成一个文件。查看文件头发现是一个jpg文件。


网上查了一下这个葫芦小金刚的英文名留下备用
GourdSmallDiamond
接下来发现c.jpg和1.mp3中都有隐写文件。
用binwalk -e 把他们都释放出来，发现这两文件是一毛一样的。而且都不能用GourdSmallDiamond解开。

使用MP3Steno解密。得到解压密码。
最终拿到flag
MSTSEC{MSTSEC_DINGANN_KEY_IS_GSD}

GreatWall

一张超大的图片。拿出StegSolve来看看吧

按老师说的在RGB的0图层都发现了不和谐的黑色。考虑是LSB隐写

试了几种不同的组合，最后在BGR层看到了不同之处。这是一个JPG文件，不过有个错误的头。

修复文件头后

这就很过份了。。在文件头上出现了这种东西。
考虑+是分隔符。—— 和 - 应当是编码。试试摩尔斯密码解密。发现不对。
考虑二进制——是1，-是0

list1 = [1010011,1110100,110011,1100111,110100,1101110,110000,1100111,1110010,110100,1110000,1101000,1111001,1011111,110001,1110011,1011111,110001,1101110,1110100,110011,1110010,110011,1110011,1110100,110001,1101110,1100111]

  result += chr(tmp)

拿到flag{St3g4n0gr4phy_1s_1nt3r3st1ng}

easy misc

这里有一个算式，写一个python脚本计算一下

import math
num = (math.sqrt(2524921)*85/5+2)/15-1794
print(num)
