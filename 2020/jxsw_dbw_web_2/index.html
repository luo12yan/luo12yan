作业三 ics-05

此题来自攻防世界。
主页非常高大上但是没有什么有用的东西，唯一能点击进入的就是这个页面
看到index.php?page=index，考虑可能是文件包含漏洞尝试使用PHP内置协议读取源码。


得到BASE64编码的index.php源文件，解码找到关键代码。

他这里使用了HTTP协议头HTTP_X_FORWARDED_FOR来判断是否是本地登录
然后使用preg_replace的/e漏洞执行PHP。

preg_replace函数原型：

特别说明：
/e 修正符使 preg_replace() 将 replacement 参数当作 PHP 代码（在适当的逆向引用替换完之后）。提示：要确保 replacement 构成一个合法的 PHP 代码字符串，否则 PHP 会在报告在包含 preg_replace() 的行中出现语法解析错误。
举例：

如果我们提交?h=phpinfo()，phpinfo()将会被执行（使用/e修饰符，preg_replace会将 replacement 参数当作 PHP 代码执行）。

那么就开始搞了。
用Burp Suite添加HTTP头并提交特殊参数
X-FORWARDED-FOR:127.0.0.1
?pat=/test/e&rep=phpinfo()&sub=test%20test

成功执行，现在开始找FLAG了
